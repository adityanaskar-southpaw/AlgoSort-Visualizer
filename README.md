# AlgoSort Visualizer

AlgoSort Visualizer is a Python-based project that demonstrates the working of sorting algorithms through real-time visualization and performance analysis. It helps in understanding algorithm design strategies by combining graphical insights with animated execution.

## Features

- 📊 Performance analysis of sorting algorithms  
- 🎬 Real-time sorting visualization  
- ⚡ Hybrid sorting approach  
- 📈 Graphs for time and comparison count  
- 🔄 Supports different input types (random, sorted, reversed)  

## Algorithms Implemented

- Bubble Sort  
- Merge Sort  
- Quick Sort  
- Hybrid Sort (Merge + Insertion)

## Technologies Used

- Python 3.x  
- Matplotlib  
- VS Code
